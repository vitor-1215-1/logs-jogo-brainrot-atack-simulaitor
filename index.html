🧠 Brainrot Attack Simulator - Guia de Instalação no Roblox Studio

📁 Estrutura de Arquivos

ReplicatedStorage/
  └── GameData (ModuleScript)       ← GameData.lua

ServerScriptService/
  ├── PlayerDataService (Script)    ← PlayerDataService.lua
  ├── RemoteHandler (Script)        ← RemoteHandler.lua
  └── AdminAbuse (Script)           ← AdminAbuse.lua

StarterPlayer/
  └── StarterPlayerScripts/
        └── GuiController (LocalScript) ← GuiController.lua

🚀 Passo a Passo

1. Abrir o Roblox Studio

Crie um novo lugar (Baseplate)

2. Instalar o GameData (ModuleScript)

Em ReplicatedStorage, crie um ModuleScript chamado GameData e cole o conteúdo de GameData.lua

3. Instalar o PlayerDataService (Script)

Em ServerScriptService, crie um Script chamado PlayerDataService e cole o conteúdo de PlayerDataService.lua. ⚠️ Mude RunContext para Server

4. Instalar o RemoteHandler (Script)

Em ServerScriptService, crie um Script chamado RemoteHandler e cole o conteúdo de RemoteHandler.lua

5. Instalar o AdminAbuse (Script)

Em ServerScriptService, crie um Script chamado AdminAbuse e cole o conteúdo de AdminAbuse.lua. Adicione seu UserId em ADMIN_IDS no topo do script

6. Instalar o GuiController (LocalScript)

Em StarterPlayer > StarterPlayerScripts, crie um LocalScript chamado GuiController e cole o conteúdo de GuiController.lua

🗺️ Construção do Mapa

O código é 100% funcional para a lógica. Para o mapa visual você precisa construir:

Tycoon

  • Crie uma base/plataforma para cada jogador
  • Nomeie os geradores como Generator_1, Generator_2, etc.
  • Adicione um botão de compra com um Part nomeado TycoonBuyButton

Esteira (Conveyor Belt)

  • Crie uma fila de partes no centro do mapa
  • A GUI já mostra os brainrots na parte inferior da tela automaticamente

Dungeon

  • Crie uma porta/portal nomeado DungeonPortal
  • Adicione uma Part separada para cada dungeon com o nome correspondente (ex: dungeon_1, dungeon_2)

NPCs

  • Crie um modelo de NPC chamado EnemyNPC em ServerStorage
  • O servidor spawna e controla via código (você pode customizar o modelo)
